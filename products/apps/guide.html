1. BASICトラッカーシリーズ

「BASICトラッカーシリーズ」は、adjustment-labが開発する
現場で使える構造的アプリ群です。
走行・回復・トレーニング・持久力の4要素を記録・分析し日々の動きを
“再現できる構造”として可視化します。

  • 🏃 Run Tracker： 走力と回復の記録
  • 🧘 Recovery Tracker： 疲労と睡眠の管理
  • 🏋️ Training Lite： 目標と行動の可視化
  • 💨 VO₂max Lite： 持久力の計測と推定

2. 使い方（基本操作）

  1. アプリ起動時に表示される「スプラッシュ画面」で概要を確認します。
  2. 「はじめる」を押すとメイン画面へ進みます。
  3. フォームに必要な項目を入力し、「記録する」ボタンで保存。
  4. 入力データは端末内のlocalStorageに保存され、自動で一覧へ反映されます。
データは端末ごとに保存されます。
ブラウザのキャッシュ削除で消去される場合があります。

3. ホーム画面への追加（PWA対応）

すべてのトラッカーはPWA（Progressive Web App）に対応しています。
ホーム画面に追加すると、スマートフォンアプリのように起動できます。

  • 📱 iPhone（Safari）：共有ボタン → 「ホーム画面に追加」
  • 🤖 Android（Chrome）：バナー表示 → 「ホーム画面に追加」

4. データの削除・リセット

各アプリの一覧にある「削除」ボタンで個別にデータを消去できます。
すべての記録をリセットしたい場合はブラウザ設定から「サイトデータの消去」を実行してください。

5. PRO版との違い

  • 📊 ベーシック版： 端末ローカル保存（個人利用向け）
  • ☁️ PRO版： クラウド保存・共有・自動分析に対応
  • 👥 チームトレーナーとのデータ連携・フィードバック機能を搭載
🚧 PRO版は現在開発中です。
正式リリース後、各トラッカー内からアップグレードできます。
